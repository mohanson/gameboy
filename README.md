Full featured GameBoy emulator. Let's dance!

```s
$ cargo run -- "roms/Boxes (PD).gb"
```

![sample.gif](./docs/sample.gif)

# Control

```
                    _n_________________
                    |_|_______________|_|
                    |  ,-------------.  |
                    | |  .---------.  | |
                    | |  |         |  | |
                    | |  |         |  | |
                    | |  |         |  | |
                    | |  |         |  | |
                    | |  `---------'  | |
                    | `---------------' |
                    |   _ GAME BOY      |
       Up           | _| |_         ,-. | ----> Z
   Left/Right <---- ||_ O _|   ,-. "._,"|
      Down          |  |_|    "._,"   A | ----> X
                    |    _  _    B      |
                    |   // //           |
                    |  // //    \\\\\\  | ----> Enter/BackSpace
                    |  `  `      \\\\\\ ,
                    |________...______,"
```

# Implemented

- [x] Item: GameBoy and GameBoy Color
- [x] CPU: The sharp LR35902
- [x] GPU
- [ ] APU: Need time to learn more
- [x] Cartridge
    - ROM ONLY
    - MBC1
    - MBC1+RAM
    - MBC1+RAM+BATTERY
    - MBC2
    - MBC2+BATTERY
    - ROM+RAM
    - ROM+RAM+BATTERY
    - MBC3+TIMER+BATTERY
    - MBC3+TIMER+RAM+BATTERY
    - MBC3
    - MBC3+RAM
    - MBC3+RAM+BATTERY
    - MBC5
    - MBC5+RAM
    - MBC5+RAM+BATTERY
    - MBC5+RUMBLE
    - MBC5+RUMBLE+RAM
    - MBC5+RUMBLE+RAM+BATTERY
    - HuC1+RAM+BATTERY
- [x] Joypad
- [x] MotherBoard
- [x] Timer
